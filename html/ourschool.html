Our School

Our School is a comprehensive school management system designed to simplify and automate administrative, academic, and communication processes within schools. The system provides a modern RESTful API hosted on Somee.com, making it accessible for integration with various front-end applications. The platform is tailored for administrators, teachers, students, and parents, offering a wide range of features for seamless school operations.

Key Features :

1. User Roles and Management Administrators: Add, update, or delete students, teachers, and parents. Assign teachers to specific classes and subjects. Manage school levels, departments, and classes. Send announcements to the entire school or specific groups. Track student attendance and issue warnings for high absentee rates. Teachers: Upload educational materials such as videos, books, and reviews. Record attendance, grades, and student performance reports. Communicate directly with students and parents. Send announcements specific to their classes. Students: Access grades and view materials uploaded by their teachers. Communicate with teachers for academic assistance. Stay informed through school announcements. Parents: Monitor their children’s academic progress and behavioral reports. Receive communication and updates from teachers and administrators. 2. Content Management Teachers can upload study materials categorized by subjects and classes. Materials include PDFs, videos, and other resources that enhance the learning process. 3. Attendance and Grades Accurate attendance tracking for all students, with warnings issued for excessive absences. Easy grade entry and visibility for both students and parents. 4. Announcements Administrators and teachers can send targeted announcements, ensuring timely communication.

Technologies Used :

ASP.NET Core: For building the API and ensuring high performance. Entity Framework Core: For efficient database management. SQL Server: The backend database hosting all school data. JWT Authentication: Ensures secure access and user role management. Swagger: Provides a user-friendly API documentation interface, making endpoint testing straightforward. Somee Hosting: The project is hosted on Somee.com, enabling live access to the API.

Project Objectives :

Simplify school management processes by digitizing key tasks. Enhance communication between students, parents, teachers, and administrators. Provide a secure, scalable, and user-friendly platform for educational institutions.

Use Cases :

Ideal for schools aiming to digitalize their operations without investing heavily in custom software. Useful for developers creating front-end applications to interface with this API. Supports both mobile and web-based educational platforms.
