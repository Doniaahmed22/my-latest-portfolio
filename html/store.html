Store

StoreAPI is a comprehensive inventory management system that provides a modern RESTful API interface. It allows developers to easily integrate with other systems or applications, such as web or mobile apps. The project offers a full range of services related to products, categories, and users, with support for security and access control.

Project Features:

1. Product Management Add new products. Update product details such as name, price, quantity, and description. Delete products from the system. View all products or a specific product using an ID. 2. Category Management Add new categories and link them to products. Update category details. Delete unused categories. Display a list of categories along with their associated products. 3. Order Management Add new orders with product details. Calculate the total price for each order. Update order status (e.g., Processing, Completed, Canceled). View order details based on the user. 4. User Management Register new user accounts. Login using credentials (Authentication). Control permissions based on user roles (e.g., Admin, Regular User). 5. Authentication & Authorization Use JWT Tokens to ensure secure access to resources. Manage roles and permissions to define allowed operations for each user. 6. Reports Generate reports on top-selling products. Analyze inventory to identify low-stock items. View order history with total sales.
